Pothole Photo file renaming script
2019-01-09 Earth Radar, Luke Twigg

Script file (PH-renamer.exe) is designed to be run from Windows.

Select .csv file of survey data downloaded from Trimble device - check pothole ID's to ensure proper numbering and no misnumbering has occured.
The .csv of survey data must be in the same folder as the photos to be renamed
Output photo files will be placed in a subfolder 'QLA_PH' of the folder where the photos are found. 
An error report log will be generated in .csv format and also placed in this folder.
